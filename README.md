# nMigen-Yosys simulation

It is another nMigen simulation backend based on yosys C++ generation.
The API is inspired by COCOTB

**status**: WIP


# How to use it?

```
python3 adder.py
```

# Requirements

* yosys with cxxtrl backend
* clang++
* nMigen
